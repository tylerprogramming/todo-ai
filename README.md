## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/tylerprogramming/todo-ai.git
   cd todo-ai
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env` file with your Supabase credentials

4. Start the development server:
   ```bash
   npm run dev
   ```
